Ця програма структурована, тобто є головний файл проєкту в який імпортовано модулі, в яких описані функції.
Один файл це просто документ з текстом
Інший, тобто __init__ для того щоб легше імпортуватити файли
Ще один файл для опису функції, яка рандомно обирає рядок з текстом і повертає його
Результатом програми є запит у користувача імені та безскінечним цикло While(true) користувача запитує 
чи не бажає він продовжити виконання програми, якщо користувач ввід відповідне слово про закінченння програми
вона закінчується.
